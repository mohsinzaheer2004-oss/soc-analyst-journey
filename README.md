# soc-analyst-journey

Documenting my journey to becoming a SOC Analyst — home lab builds, hands-on
writeups, and notes, logged week by week as I work through a structured
roadmap (Linux & networking fundamentals → security concepts → SOC tools →
real-world alert triage).

## Connect
- LinkedIn: [www.linkedin.com/in/mohsin-zaheer-471b643b2]

## Structure
- `home-lab/` — weekly logs of my home lab setup and practice sessions
- `tryhackme-writeups/` — writeups from TryHackMe free rooms
- `letsdefend-cases/` — alert triage case notes from LetsDefend
- `scripts/` — small Python/PowerShell scripts as I learn automation

## Roadmap I'm following
Phase 0: Computer basics → Phase 1: Networking → Phase 2: Security
fundamentals → Phase 3: SOC-specific skills (SIEM, log analysis, IR) →
Phase 4: Real SOC simulation (LetsDefend, CyberDefenders, BTLO)

Every entry follows the same format: what the alert/scenario was, what I
investigated and how, what I concluded, and what I'd recommend next.
